About

Este aplicativo funciona como um gerenciador financeiro. Inicialmente o usuário deve criar pelo menos uma conta para receber os registros de despesas e receitas. Para cada registro de receitas e despesas, o usuário especifica a data, conta, categoria e subcategoria oriundas de uma lista, valor e uma observação (falcultativo). Após alimentar o banco de dados, há opção da geração de relatórios sobre o saldo de contas, fluxo de caixa, despesas por categoria e subcategoria.

Recursos utilizados
  • Sqlite3
  • Python (Flask)
  • CSS (Bootstrap)
  • JavaScript
Nota

Este aplicativo foi desenvolvido para fins pessoais, com objetivo de aprender o funcionamento de banco de dados, front-end, back-end, python, git, lógica de programação, etc. Maiores informações em AQUI.
